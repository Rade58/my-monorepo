# Component composition

# `<slot />` element

**YOU CAN THINK OF THIS THE SAME WAS AS YOU THINK OF `children` PROP IN REACT**

`Card.svelte`

```html
<div class="card">
	<!-- slot goes here -->
	<slot />
</div>

<style>
  /* some scard styles */
</style>

```

`App.svelte`

```html
<script>
	import Card from './Card.svelte';
</script>

<main>
	<Card>
		<!-- content goes here -->
		<span>John Wayne</span>
		<span>Secretary Of State</span>
	</Card>
</main>

<style>
	/* ***** */
</style>
```
