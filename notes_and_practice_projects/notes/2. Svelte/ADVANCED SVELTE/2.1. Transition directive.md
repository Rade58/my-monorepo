# Transitions directive

```html
<script>
	import {fade} from 'svelte/transition'
	let visible = true;
</script>

<label>
	<input type="checkbox" bind:checked={visible} />
	visible
</label>

<!-- as you can see it is a directive -->

{#if visible}
	<p transition:fade>
		Fades in and out
	</p>
{/if}

```
